# 📑 คู่มือการตรวจสอบคุณภาพงานแปล (Translation QA Checklist)

ไฟล์นี้สร้างขึ้นเพื่อใช้ตรวจสอบความเรียบร้อยของงานแปล โดยอ้างอิงจากขั้นตอนการตรวจสอบ (Automated Checks) ในระบบ

## 1. ความถูกต้องของไฟล์และโฟลเดอร์ (File & Folder Integrity)
- [ ] ตรวจสอบว่าไม่มีโฟลเดอร์ที่ซ้ำซ้อนกัน (Checking DUP folders)
- [ ] ตรวจสอบว่าไม่มีไฟล์ที่ซ้ำซ้อนกัน (Checking DUP files)
- [ ] โครงสร้างโฟลเดอร์ถูกต้องตามที่กำหนดไว้ (Verify folder layout)
- [ ] ตรวจสอบไฟล์ที่ยังไม่มีการเปลี่ยนแปลง เพื่อดูว่ามีส่วนไหนตกหล่นหรือไม่ (Looking for unchanged Files)

## 2. การเข้ารหัสและรูปแบบไฟล์ (Encoding & Formatting)
- [ ] ไฟล์ CSV ทั้งหมดต้องเข้ารหัสเป็น **UTF-8** (Verify CSVs as UTF-8)
- [ ] ไฟล์ CSV มีการใช้ตัวอักษรขึ้นบรรทัดใหม่ (EOL) ที่ถูกต้อง (Verify CSVs with an EOL)
- [ ] ไฟล์ CSV ต้อง **ไม่มี BOM** (Verify CSVs without BOM)
- [ ] โครงสร้างและไวยากรณ์ของไฟล์ CSV ถูกต้อง ไม่แตก (Verify CSVs)
- [ ] ไม่มีปัญหาการใช้เครื่องหมายคำพูดผิดพลาด (Checking CSVs for quote issues)
- [ ] ไม่มีบรรทัดที่ข้อมูลซ้ำซ้อนกัน (Check for dup lines)
- [ ] This diff contains a change in line endings from 'CRLF' to 'LF'.

## 3. การตรวจสอบเฉพาะของเกม (PSO2 Specific Checks)
- [ ] ไฟล์ CSV มีโครงสร้างตรงตามรูปแบบที่ระบบเกมต้องการ (Verifying CSVs for PSO2 format)
- [ ] ตรวจสอบความถูกต้องของการใช้คำสั่งต่างๆ ภายในข้อความ เช่น การขึ้นบรรทัดใหม่ หรือ Tag สี (Verifying command usages)
- [ ] ไม่มีตัวอักษรหรือข้อความแปลกปลอมในชื่อของ NPC (Check ODD strings in NPC names)
- [ ] ฟอนต์ในเกมสามารถแสดงผลตัวอักษรที่ใช้แปลได้ครบถ้วน ไม่เป็นสี่เหลี่ยม (Verify Font rendering)

## 4. การสะกดคำและคุณภาพภาษา (Spelling & Quality)
- [ ] ดำเนินการตรวจคำผิดในไฟล์ CSV (Spell check CSVs)
- [ ] จัดการแก้ไขคำผิดที่ระบบตรวจพบให้เรียบร้อย (Sort out spelling mistakes)
- [ ] ตรวจทานรอบสุดท้ายว่าไม่มีคำผิดหลงเหลืออยู่ (Are there any spelling mistakes)
