# 📝 Standard Commit Messages for Thai Translation (NEKO★FAMILY)

คู่มือรวมประโยค Commit แบบมาตรฐาน สำหรับระบุรายละเอียดการอัปเดตงานแปล เพื่อให้ AI และทีมงานสามารถตรวจสอบประวัติการทำงาน (History) ได้อย่างรวดเร็วและเป็นระเบียบ

## 1. 🌐 Translation Updates (การอัปเดตและเพิ่มคำแปล)

ใช้หมวดหมู่นี้เมื่อมีการแปลเนื้อหาใหม่ หรืออัปเดตเนื้อหาเดิม

* **`Update Thai translation for ui_tips_reb.csv`**
    * *ความหมาย:* อัปเดตงานแปลภาษาไทยในไฟล์นี้ (มีการแก้ไขหรืออัปเดตคำแปลจากเดิม)
* **`Add Thai translation to ui_tips_reb.csv`**
    * *ความหมาย:* เพิ่มคำแปลภาษาไทยเข้าไปในไฟล์นี้ (แปลเนื้อหาใหม่ที่ยังไม่เคยมีภาษาไทยมาก่อน)

## 2. 🛠️ Bug Fixes & Formatting (การแก้บั๊กและโครงสร้างไฟล์)

ใช้หมวดหมู่นี้เมื่อมีการแก้ปัญหาทางเทคนิค เพื่อให้ระบบ (CircleCI) สามารถตรวจสอบผ่านได้

* **`Fix missing quotes in ui_tips_reb.csv`**
    * *ความหมาย:* แก้ปัญหาเครื่องหมายฟันหนู (" ") ขาดหาย หรือใส่ไม่ครบตามรูปแบบของ CSV
* **`Fix formatting issues in ui_tips_reb.csv`**
    * *ความหมาย:* แก้ปัญหาโครงสร้างไฟล์พัง เช่น การเผลอขึ้นบรรทัดใหม่ผิดที่ (Extra line) หรือช่องว่างผิดปกติ

## 3. ✨ Proofreading & Polishing (การขัดเกลาและแก้ไขคำผิด)

ใช้หมวดหมู่นี้สำหรับการตรวจทาน ปรับปรุงภาษา และแก้ไขคำผิดเล็กๆ น้อยๆ

* **`Fix typo in ui_tips_reb.csv`**
    * *ความหมาย:* แก้ไขคำสะกดผิด หรือพิมพ์ตกหล่นเล็กๆ น้อยๆ 
* **`Refine text formatting and polishing in ui_tips_reb.csv`**
    * *ความหมาย:* จัดรูปแบบข้อความใหม่และขัดเกลาสำนวนคำแปลให้สละสลวย เป็นธรรมชาติมากขึ้น

## 4. ⏪ Reverts & Restorations (การกู้คืนและย้อนกลับไฟล์)

ใช้หมวดหมู่นี้เมื่อเกิดข้อผิดพลาดร้ายแรง และต้องการนำไฟล์ต้นฉบับมาเขียนทับ

* **`Revert ui_tips_reb.csv to original base file`**
    * *ความหมาย:* ยกเลิกการแก้ไขทั้งหมด และย้อนกลับไปใช้สถานะของไฟล์ต้นฉบับ
* **`Restore original ui_tips_reb.csv`**
    * *ความหมาย:* กู้คืนไฟล์ต้นฉบับ (นำไฟล์จากฝั่ง JP/EN มาวางทับเพื่อแก้ปัญหาไฟล์พังหรือสูญหาย)

## 5. 📏 UI & Text Fitting (การแก้ปัญหาข้อความล้นจอและจัดหน้า)

ใช้หมวดหมู่นี้เมื่อนำแพตช์ไปทดสอบในเกมแล้วพบว่าข้อความแปลยาวเกินไป ทับซ้อนกัน หรือตัดบรรทัดไม่สวยงาม

* **`Fix text overflow in [filename].csv`**
    * *ความหมาย:* แก้ปัญหาข้อความแปลยาวเกินไปจนแสดงผลล้นกรอบหน้าต่าง (UI) ของเกม
* **`Adjust line breaks in [filename].csv`**
    * *ความหมาย:* ปรับแก้ตำแหน่งการเว้นวรรค หรือจุดขึ้นบรรทัดใหม่ (`<br>`) ให้อ่านง่ายและไม่ฉีกคำ
* **`Fix broken tags in [filename].csv`**
    * *ความหมาย:* แก้ไขแท็กสีหรือตัวแปรระบบเกมที่พัง/สูญหายระหว่างการแปล (เช่น แท็ก `<c>`, `{0}`)

## 6. 📖 Terminology & Consistency (คำศัพท์เฉพาะและความสม่ำเสมอ)

ใช้หมวดหมู่นี้เมื่อมีการอัปเดตตกลงเปลี่ยนคำศัพท์ของทีม หรือต้องการไล่แก้คำให้เป็นมาตรฐานเดียวกัน

* **`Update terminology in [filename].csv`**
    * *ความหมาย:* ปรับคำแปลศัพท์เฉพาะ (เช่น ชื่อสเตตัส, ระบบเกม) ให้ตรงกับพจนานุกรม (Glossary) ล่าสุดของทีม
* **`Standardize translation across multiple files`**
    * *ความหมาย:* ไล่แก้คำแปลของคำใดคำหนึ่ง ให้ตรงกันเป็นมาตรฐานเดียวในหลายๆ ไฟล์พร้อมกัน

## 7. 🔄 Patch Syncing & Game Updates (การซิงก์ไฟล์และอัปเดตตามแพตช์)

ใช้หมวดหมู่นี้เมื่อเซิร์ฟเวอร์หลัก (JP/Global) มีการอัปเดตแพตช์เวอร์ชันใหม่ และเราต้องอัปเดตไฟล์แปลตาม

* **`Sync [filename].csv with the latest JP patch`**
    * *ความหมาย:* ซิงก์โครงสร้างไฟล์ให้ตรงกับแพตช์ล่าสุด (ดึงโค้ดบรรทัดใหม่ที่เกมเพิ่งเพิ่มเข้ามาใส่เตรียมไว้)
* **`Translate new strings for [Patch Name] update`**
    * *ความหมาย:* เพิ่มคำแปลสำหรับข้อความ/ไอเทมใหม่ ที่เพิ่งถูกเพิ่มเข้ามาในแพตช์ล่าสุด
* **`Remove unused strings from [filename].csv`**
    * *ความหมาย:* ทำความสะอาดไฟล์ ลบบรรทัดข้อความเก่าหรือข้อความขยะที่ระบบเกมไม่ได้ใช้งานแล้วทิ้งไป
