### นโยบายและแนวทางการใช้ระบบ AI ในกระบวนการแปล (Translation Guideline)

เรียน ทีมงานนักแปลทุกท่าน

เพื่อรักษามาตรฐาน คุณภาพ และความเป็นเอกภาพของงานแปล (Consistency) ทางส่วนงานจึงขอความร่วมมือจากทุกท่านปฏิบัติตามแนวทางการใช้เทคโนโลยี AI ในการช่วยแปล ดังต่อไปนี้ครับ:

1. **การรักษามาตรฐานโครงสร้างข้อมูล (Markdown):** 
   กรุณาใช้รูปแบบ Markdown (MD) เดียวกันตามที่กำหนดไว้ในต้นฉบับอย่างเคร่งครัด เพื่อให้เนื้อหาและรูปแบบการจัดวาง (Formatting) เป็นไปในทิศทางเดียวกันทั้งหมด

2. **ขอบเขตการทำงานอย่างจำกัด (Strict Scope):** 
   อนุญาตให้ใช้ AI ช่วยประมวลผล**เฉพาะในส่วนเนื้อหาที่ได้รับมอบหมายเท่านั้น** ห้ามมิให้ระบบ AI หรือตัวผู้แปลทำการแก้ไข ดัดแปลง หรือเขียนทับ (Overwrite) ในส่วนที่เป็นผลงานของนักแปลท่านอื่นโดยเด็ดขาด

3. **กระบวนการทบทวนและตรวจสอบกรณีมีความเห็นแย้ง (Dispute Resolution):** 
   หากพบข้อผิดพลาดในส่วนอื่น หรือมีความเห็นแย้งเกี่ยวกับสำนวนภาษาและคำศัพท์ **กรุณาถ่ายภาพหน้าจอ (Screenshot) พร้อมระบุจุดที่ต้องการแก้ไขลงในกลุ่มสื่อสารส่วนกลาง** เพื่อให้ทีมงานร่วมกันพิจารณาก่อนดำเนินการใดๆ 

ทางส่วนงานหวังเป็นอย่างยิ่งว่าจะได้รับความร่วมมือจากทุกท่าน เพื่อร่วมกันสร้างสรรค์ผลงานที่มีคุณภาพสูงสุดครับ
